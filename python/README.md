# Python calculator

## Установка
Windows/macOS: нужен Python 3.10+ (проверить `python --version` или `python3 --version`).

## Запуск тестов
Windows:
```bat
python -m unittest -v
```

macOS:
```bash
python3 -m unittest -v
```

## Что это демонстрирует
- Простые функции в одном модуле.
- Встроенный `unittest` без внешних зависимостей.
- Исключение `ZeroDivisionError` при делении на ноль.
